# Aşama 6 - Güncelleme Komutları (updateA, updateC)

## Hedef
Uygulama ve CLI güncelleme komutlarını implement etmek.

## Yapılacaklar

### 6.1 `updateA` Komutu
- GitHub API ile uygulamanın en son sürümünü kontrol eder
- Mevcut sürüm ile karşılaştırır
- En son sürümü indirir
- Mevcut uygulamanın üstüne kurar
- İndirme ilerlemesi gösterir
- Başarılı/başarısız geri bildirim mesajı
- `--info`, `--help`, `--stime` parametre desteği

### 6.2 `updateC` Komutu
- GitHub API ile CLI'ın en son sürümünü kontrol eder
- Mevcut sürüm ile karşılaştırır
- En son sürümü indirir
- Mevcut CLI'ın üstüne kurar
- İndirme ilerlemesi gösterir
- Başarılı/başarısız geri bildirim mesajı
- `--info`, `--help`, `--stime` parametre desteği

### 6.3 GitHub Entegrasyonu
- GitHub Releases API bağlantısı
- Sürüm karşılaştırma mantığı
- Dosya indirme ve kurma mekanizması
