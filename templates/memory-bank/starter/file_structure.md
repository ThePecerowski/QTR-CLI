## Klasör Açıklamaları

| Klasör | Amaç |
|--------|-------|
| `pages/` | URL'ye doğrudan karşılık gelen PHP dosyaları |
| `routes/` | Web ve API yönlendirme dosyaları |
| `app/` | Controller, model ve core mantık |
| `resources/views/` | HTML şablonları (layout, partial, component) |
| `database/` | SQL dosyaları |
| `storage/` | Log, backup ve geçici dosyalar |
| `public/` | Kamuya açık giriş noktası |
