# Project - Parametreler (SpeakerQuarter CLI)

> CLI komutlarının parametreleri ve kullanım şekilleri burada anlatılır.

## Genel Parametreler

| Parametre | Açıklama |
|-----------|----------|
| `--info` | Komutun ne işe yaradığını söyler |
| `--help` | Komutun hata vermesi durumunda ne yapılabileceğini söyler |
| `--stime <süre>` | Belirlenen süre sonra komutu çalıştırır |

## Özel Parametreler

### fix komutu
| Parametre | Açıklama |
|-----------|----------|
| `--doctor` | Windows'da çalışmaya engel bir durum var mı kontrol eder |
| `--show` | Mevcut sorunların çözümü hakkında bilgi verir |
